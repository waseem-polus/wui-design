# Link
 
## Icons
if the ``showIcons`` prop is set to ``true``, a link will display the following icons if:
|Icon|Name|Pattern|
|----|----|-------|
|<Icon name="Mail" color="#000" />|[Mail](https://lucide.dev/icons/mail)|Link starts with "mailto:"|
|<Icon name="Phone" color="#000" />|[Phone](https://lucide.dev/icons/phone)|Link starts with "tel:"|
|<Icon name="ExternalLink" color="#000" />|[External Link](https://lucide.dev/icons/external-link)|Link starts with "http[s]://www"|

## API
|Prop|Type|Default|Description|
|----|----|-------|-----------|
|``children``|string||The text the hyperlink will display|
|``link``|string||The link the hyperlink will lead to|
|``showIcons``|boolean|`true`|Show icons for mail, phone, and external|
