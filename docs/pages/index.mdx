# How to Install?

The Design installation of wui offers opinionated, pre-styled components.

<TabGroup>
    <Tab label="npm">        
        ``` bash copy
        npm i @wysteria/design react react-dom
        ```
    </Tab>

    <Tab label="yarn">
        ``` bash copy
        yarn add @wysteria/design react react-dom
        ```
    </Tab>

    <Tab label="pnpm">        
        ``` bash copy
        pnpm add @wysteria/design react react-dom
        ```
    </Tab>
</TabGroup>

## Peer Dependencies
The project depends on the following dependencies to be included in your project:
``` json
react: "^18",
react-dom: "^18"
```
