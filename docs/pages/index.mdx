# How to Install?

The Design installation of wui offers opinionated, pre-styled components.

import { Tabs } from 'nextra/components'
 
<Tabs items={['npm', 'yarn']}>
    <Tabs.Tab>        
        ``` bash copy
        npm install @willoui/design
        ```
    </Tabs.Tab>

    <Tabs.Tab>
        ``` bash copy
        yarn add @willoui/design
        ```
    </Tabs.Tab>
</Tabs>

## Peer Dependencies
The project depends on the following dependencies to be included in your project:
``` json
react: "^18",
react-dom: "^18"
```
